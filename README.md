# heisenberg-2-ppt-automation-tool
Streamlining church presentations: A Python-based automation tool to scrape song lyrics and generate formatted PowerPoint slide decks in seconds.
Heisenberg 2.0 🎵
A powerful, Python-based automation tool designed to generate song lyrics PowerPoint presentations effortlessly.

📖 Overview
Heisenberg 2.0 was built to streamline the presentation creation process for church services. By automating the extraction of lyrics and the construction of slides, it removes the manual drudgery of formatting song presentations in Tamil and English.

⚙️ Technical Stack
Backend: Python, Selenium (Data Scraping), PyQt6 (GUI Framework)

Frontend: HTML5/CSS3 (WebEngineView)

Packaging: Inno Setup (Windows Installer)

🚀 Key Features
Automated Extraction: Scrapes lyrics directly from TamilChristianSongs.

Smart UI: A sleek, web-based interface that handles real-time status updates.

System Independent: Fully portable design with registry-based configuration persistence.

Seamless Generation: One-click PowerPoint creation with automatic file opening.

🛠️ Implementation Details
The process is split into two robust components:

Data Scraping: Utilizes Selenium to fetch lyrics. The system is designed to handle edge cases in web data, with modular code in data_scraping.py.

Presentation Synthesis: Orchestrates the creation of slides. Future updates are focused on making this layer fully system-agnostic for universal deployment.

📥 How to Install
Download the latest installer from the Releases page.

Run the mysetup.exe file.

The application will be automatically added to your Start Menu for easy access.
